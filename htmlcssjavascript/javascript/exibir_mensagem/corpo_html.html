Modificar texto dentro de uma tagname

Ao usarmos o código - innerText - podemos pegar o texto que foi escrito entre um elemento. Se o elemento for um conteinêr de outros elementos, todos os textos contidos, de onde pedimos, serão mostrados

Este parágrafo será modificado ao rodar o script

window.alert(document.getElementById('teste_1').innerText) window.alert(document.getElementById('teste_2').innerText) document.getElementById('teste_3').innerText = 'Este parágrafo foi modificado ao rodar o script'

O que for passado de valor para o - innerText - é interpretado apenas como texto, logo o sinal de maior e menor não gerariam o efeito de criar uma tagname

Este parágrafo será alterado por javascript para mostrar a tagname - span -:

Agora caso queira mudar o texto dentro do elemento com efeito html, ou pegar um elemento podendo ver as tagnames no conteiner usamos o - innerHTML -
