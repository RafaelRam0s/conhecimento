Alerta simples

Com este código podemos criar um aviso para o usuário, contudo tenha cuidado, pois os alertas possuem escopo global, isto é, antes de um código ser lido ou executar uma ação, o alerta é feito antes (perde para variável - var - em precedência de escopo)

window.alert("Hello World!")

Alerta com opção

Este alerta oferece duas opções, se você armazenar o código em uma variável assim que o usuário clica em - OK - é retornado o valor true, mas caso ele selecione - Cancelar - é retornado false.

window.confirm("Você gerou um script!") let escolha = window.confirm("Selecione uma das opções deste alerta e veja o que retorna") window.alert("O que você selecionou retornou: " + escolha)

Alerta com Pergunta

Este alerta é usado para poder captar uma resposta, se nada for escrito é retonado vazio, se algo for escrito é retornado o que foi preenchido e se clicar em cancelar é retornado null

let nome = window.prompt("Qual o seu nome?") window.alert(nome)
