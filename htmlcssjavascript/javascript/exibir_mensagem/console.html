Escrever no console

Console.log()

Ao "inspecionar o elemento" da página, possuimos o campo console, nesse campo é possível programarmos em javascript diretamente na página, além de podermos programar é possível passar mensagens nessa tela, para isso usamos o código - console.log() - o valor que estiver dentro do parênteses será aparecido como uma mensagem

console.log("Olá Mundo!"); console.log(123); console.log(true); let apresentacao = "Hello World!"; console.log(apresentacao);

Abra o console e veja que as mensagens foram apresentadas

Este recurso só é recomendável de ser usado enquanto estiver fazendo testes na aplicação, quando for colocar o seu site na rede, retire-os

@@@@@ escrever uma informação

console.info("Olá Mundo!"); console.info(123); console.info(true); let apresentacao = "Hello World!"; console.info(apresentacao);

Limpar o console

No console são passadas informações de quando tem algum erro na página e em alguns casos, utilizamos códigos que escrevem nele. Caso queira limpar o console utilize o código - console.clear() -

console.clear()

Aviso no console

Mandar avisos pelo console é uma ótima maneira de manter você alerta na sua aplicação. Temos dois níveis de perigo na mensagem do console, o aviso e o erro, o aviso é só para avisar que algo pode acontecer e que você deve ficar esperto em futuras atualizações, já o erro é caso algo na sua aplicação não esteja funcionando

console.warn("Este é um aviso de alerta") console.error("Este é um aviso de erro")

Agrupar mensagens no console

É possível organizarmos as mensagens no console através de grupos, desta forma podemos englobar uma série de mensagens dentro de um bloco para facilitar a visão e entendimento, para isso usamos o - console.group() - para iniciar uma seção e o console.groupEnd() - para finalizar a seção. Se dentro do parenteses de - console.group() - for informado um valor, ele será o nome do grupo, esse nome serve meramente ilustrativo para aparecer no console e te ajudar a se organizar, não age como um indentificador. Se colocarmos algum valor dentro do parênteses de - console.groupEnd() - servirá apenas para o programador na leitura do código se guiar, pois o valor dentro dele não aparece no console, não é referenciado e nem faz uma referência. O - console.groupEnd() - fecha o último grupo que foi escrito

O - console.group() - cria um grupo já aberto, ai precisamos clicar na setinha do console para minimizar, mas caso queiramos iniciar um grupo já minimizado, usamos o código - console.groupCollapsed() - que tem o mesmo funcionamento do - console.group() - tirando a parte que inicia minimizado

console.group() console.log("teste 1") console.groupEnd() console.group("Grupo 2") console.log("teste 1") console.group("Grupo 3") console.log("teste 1") console.groupEnd("Fechamento do grupo 3") console.log("teste 1") console.groupEnd("Fechamento do grupo 2") console.groupCollapsed("Grupo 4") console.log("teste 1") console.group("Grupo 5") console.log("teste 1") console.groupEnd("Fechamento do grupo 5") console.log("teste 1") console.groupEnd("Fechamento do grupo 4") console.log("teste 1")
