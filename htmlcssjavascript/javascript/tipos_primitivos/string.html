O que é uma string

String é toda informação textual que temos, por exemplo o nome de uma pessoa, para colocarmos algo como string temos que colocar entre "aspas simples", "aspas duplas" ou "crases"

function demonstracaoString() { let nome = 'Rafael Ramos' window.alert("Olá " + nome) }

@@@@@@@Uso de códigos dentro das strings

O "contra-barra" dentro de uma string leva a próxima letra após ele como literal, então se criarmos uma string por aspas simples e dentro dela quisermos colocar uma ou mais aspa simples devemos usar contra-barra e em seguida colocar o sinal de aspa simples. Contudo algumas letras em conjunto com o contra-barra servem para fazer algumas funções textuais como por exemplo a quebra de linha, segue alguns usos com letra:

Usos do "contra-barra" \b dá um espaço, \f , \n quebra de linha, \r, \t, \v

Além do contra-barra no entre crases tem outra série de símbolos que nos permite ter outra ação. Essa acão seria abrir um espaço no meio do da string para gerar um script, mas ao terminar o script continua dentro da string, seria igual a escrever - window.alert('Olá ' + 'Rafael' + ', seja bem-vindo') - na forma com código; - window.alert(`Olá ${'Rafael'}, seja bem-vindo`) -

function codigosNaString() { window.alert('Texto para testes de código com o \'contra-barra\': \n \\n faz a quebra de linha, @@@@@') window.alert(`Olá ${'Rafael'}, seja bem-vindo`) }

Encontrar uma string em uma string

Para procurar por uma frase ou palavra no meio de uma string, podemos usar o - indexOf() - onde dentro do parênteses deve ser informado o que você quer procurar

Caso o parâmetro informado não seja encontrado dentro da string é retornado o valor -1

Caso seja encontrado o valor o resultado vai ser a posição em que a letra está, sendo que a contagem da primeira posição é 0, a da segunda é 1, da terceira 2 e assim por diante em uma PA.

Só é procurado pela primeira ocorrência, se tiver na frase duas palavras iguais e vc buscar por elas somente a primeira ocorrência é procurada

Ao passarmos um segundo parâmetro do tipo number natural, podemos informar a partir de que posição poderá ser iniciada a busca.

Para retirar um pedaço de texto de uma string

Temos 3 formas de fazer o recorte de uma string, por slice(), substring() e substr()

- slice() - o primeiro parâmetro informa a partir de que posição será feito feito o recorte e o segundo parâmetro informa até que posição deve ir. Os parâmetros devem ser números inteiros. O primeiro parâmetro tem que ser menor que o segundo para ser retorndo algo diferente de string vazia - '' - . Se for omitida o segundo parâmetro é feito o recorte do restante da string. Se o valor inicial for maior do que a quantidade de caracteres existentes será retornado uma string vazia

function recortar() { let frase_qualquer = 'Trecho para exemplo/testes.' window.alert('Vamos usar a seguinte frase para exemplo: ' + frase_qualquer + '\n A seguir vamos recortar da posição 2 até a 5, resultado: ' + frase_qualquer.slice(2, 5) + '\n A seguir vamos recortar da posição 2 até a -5, resultado: ' + frase_qualquer.slice(2, -5) + '\n A seguir vamos recortar da posição -5 até a -2, resultado: ' + frase_qualquer.slice(-5, -2) + '\n A seguir vamos recortar da posição -5 até a -5, resultado: ' + frase_qualquer.slice(-5, -5) + '\n A seguir vamos recortar da posição 5 até a 2, resultado: ' + frase_qualquer.slice(5, 2) + '\n A seguir vamos recortar da posição 5 até o final, resultado: ' + frase_qualquer.slice(5) + '\n A seguir vamos recortar da posição -5 até o final, resultado: ' + frase_qualquer.slice(-5) + '\n A seguir vamos recortar da posição 1000 até o final, resultado: ' + frase_qualquer.slice(1000)) }

- substring() - é igual ao slice exceto pelo fato de ele não poder receber o primeiro parâmetro negativo e se o segundo parâmetro for negativo ele recorta até o início

function recorte() { let frase_qualquer = 'Trecho para exemplo/testes.' window.alert('Vamos usar a seguinte frase para exemplo: ' + frase_qualquer + '\n A seguir vamos recortar da posição 2 até a 5, resultado: ' + frase_qualquer.substring(2, 5) + '\n A seguir vamos recortar da posição 2 até a -5, resultado: ' + frase_qualquer.substring(2, -5) + '\n A seguir vamos recortar da posição -5 até a -2, resultado: ' + frase_qualquer.substring(-5, -2) + '\n A seguir vamos recortar da posição -5 até a -5, resultado: ' + frase_qualquer.substring(-5, -5) + '\n A seguir vamos recortar da posição 5 até a 2, resultado: ' + frase_qualquer.substring(5, 2) + '\n A seguir vamos recortar da posição 5 até o final, resultado: ' + frase_qualquer.substring(5) + '\n A seguir vamos recortar da posição -5 até o final, resultado: ' + frase_qualquer.substring(-5) + '\n A seguir vamos recortar da posição 1000 até o final, resultado: ' + frase_qualquer.substring(1000)) }

- substr() - também faz recorte, mas age diferente do slice(). Ele só aceita no primeiro parâmetro numéricos inteiros e no segundo numéricos naturais. O primeiro parâmetro informa de onde deve ser iniciado o recorte e o segundo parâmetro informa quantas casas à frente do primeiro parâmetro devem ser recortadas, se omitido vai até o final

function recorteEmFrente() { let frase_qualquer = 'Trecho para exemplo/testes.' window.alert('Vamos usar a seguinte frase para exemplo: ' + frase_qualquer + '\n A seguir vamos recortar da posição 2 os próximos 5 caracteres, resultado: ' + frase_qualquer.substr(2, 5) + '\n A seguir vamos recortar da posição 5 os próximos 2 caracteres, resultado: ' + frase_qualquer.substr(5, 2) + '\n A seguir vamos recortar da posição 5 até o final, resultado: ' + frase_qualquer.substr(5) + '\n A seguir vamos recortar da posição 1000 até o final, resultado: ' + frase_qualquer.substr(1000) + '\n A seguir vamos recortar da posição -5 os próximos 2 caracteres, resultado: ' + frase_qualquer.substr(-5, 2) + '\n A seguir vamos recortar da posição 5 os próximos -2 caracteres, resultado: ' + frase_qualquer.substr(5, -2)) }

Substituir um trecho por outro em uma String

O - replace() -, usa dois parâmetros, o primeiro deve ser informada a string que deseja procurar ou uma forma regex de busca, e no segundo deve-se informar por qual string deseja substituir.

Para tranformar toda uma string em minúculo ou em maiúsculo

Para tranformar todo um texto em minúculo, usamos - toLowerCase() - e para tranformar todo um texto em maiúsculo, usamos - toUpperCase() -

function transformarTexto() { let frase_qualquer = 'Trecho para exemplo/testes.' window.alert('Vamos usar a seguinte frase para exemplo: ' + frase_qualquer + '\n Tranformando o texto em minúsculo: ' + frase_qualquer.toLowerCase() + '\n Tranformando o texto em maiúculo: ' + frase_qualquer.toUpperCase() ) }

Pegar um caracter de uma string

Quando queremos pegar somente um caracter em alguma posição de uma string temos, as seguintes formas:

Por acesso à propriedade - string[numberofposition] - ao chamarmos assim é como se tratassemos cada caracter como um item de uma lista, mas a pesar do acesso ser assim, não podemos armazenar ou substituir um caracter de forma simples colocando um recebe

function pegarCaracterPorPropriedade() { let str = "HELLO WORLD"; str[0] = "A"; // Não dá erro, mas não funciona alert(str[0]) }

Por - .charAt(position) - Em que é retornado a letra em uma posição da string sendo que a primeira é 0, a segunda é 1 e assim por diante

let str = "Olá Mundo"; alert(str.charAt(0))

Por - .charCodeAt(position) - faz as mesmas coisas e da mesma forma que o charAt() a diferença é que o valor retornado é em código ASCII

var str = "HELLO WORLD"; str.charCodeAt(0); // returns 72
