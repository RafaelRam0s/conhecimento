Para que serve o Object

Imaginemos que queiramos guardar todas as informações de um carro dentro de cada variável; - var carroNome = 'Fiat patrocina nóis', carroPortas = 4, carroCor = 'branco' - perceba que temos que usar muitas variáveis para armazenar os dados, do jeito que escrevi, estão todas organizadas e possuem um padrão de começar com a palavra "carro", mas e se eu colocasse somente; - var nome = 'Fiat patrocina nóis', portas = 4, cor = 'branco' - . Veja que colocamos em variáveis os dados, contudo parecem muito soltas e provavelmente você precisará de um comentário para lembrar que esses dados estão se tratando de um carro.

Pensando nisso temos o object, ele basicamente fará uma variável possuir diversas subdivisões e essas subdivisões irão agir como outras variáveis

function dadosDeUmCarro() { let carro = { modelo: 'Fiat patrocina nóis', portas: 4, cor: 'branco', motor: { cavalos: 200, cilindradas: 50 } } window.alert(`O carro de modelo ${carro.modelo}, de ${carro.portas} portas, é da cor ${carro.cor} e possui um motor com ${carro.motor.cavalos} cavalos rodando à ${carro.motor.cilindradas} cilindradas`) }

Como criar um Object

Um objeto é criado com a abertura e fechamento das "chaves", tudo o que estiver entre chaves será o objeto, a próxima palavra que for escolhida terá apenas escopo local então se tivermos uma variável x definida e depois criarmos um objeto com nome x, está tudo bem esses dois x serão diferentes na hora de chamar-los. Após escolher uma palavra chave é utilizado o "sinal de dois pontos" que age parecido com um "recebe" de uma variável e por último você coloca o dado que aquela propriedade da variável irá receber.

Os objetos podem ser, tanto criado em apenas uma linha quanto em várias, segue exemplos:

var person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"}; var person = { firstName: "John", lastName: "Doe", age: 50, eyeColor: "blue" };

Como usar um object

Existem duas formas de chamadas, vou explicar a que mais prefiro, você tem que colocar o nome da variável que está armazenando o objeto (objectName), e depois colocar o "ponto final" e o nome do objeto que você quer acessar (propertyName), se esse último tem outro objeto armazenado, use o ponto final e o nome da próxima propriedade para acessar

objectName.propertyName objectName["propertyName"]

Usando uma função armazenada em um objeto

É possível armazenar uma função dentro do objeto, mas para acessa-lo é um pouco diferente, segue exemplo de como montar:

var person = { firstName: "John", lastName : "Doe", id : 5566, fullName : function() { return this.firstName + " " + this.lastName; } };

E para chamar essa função não é nada tão complexo:

name = person.fullName(); name = person.fullName;

A diferença é que no primeiro caso você executa a função, pois colocou o abre e fecha parênteses, no segundo o que ocorre é que você mostrará exatamente como a função foi digitada, mostrando assim o código
