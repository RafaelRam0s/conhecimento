@@@@@Estruturas de repetição

A repetição serve para não precisarmos ficar escrevendo um mesmo código mil vezes, por exemplo imagine que eu queira mandar dois alertas iguais eu teria que escrever o mesmo código 2 vezes, mas podemos fazer com que tenhamos um código que basta informar até quando queremos que ele se repita:

while(){} - enquanto o que estiver no parênteses for verdadeiro o bloco irá se repetir do{} while() - primeiro executa e depois verifica se deve realizar de novo for(criarVariaveis;enquantoTrueRepita;façaIssoACadaRepetição){} - for (umaVariavelNova in umArray) {} - para cada item no bloco o for é rodado e armazena o valor do item dentro da variável for (umaVariavelNova of umaLista) {} - para cada item na lista o for é rodado e armazena o valor do item dentro da variável
