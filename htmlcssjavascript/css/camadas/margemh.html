Margem

Margem:

Margens são espaços vazios que dão um distanciamento de um elemento até outro

Por exemplo estes parárafos que possuem uma distância entre si

Eu posso tanto aumentar (com um valor positivo)

Quanto diminuir essa distância (com um valor negativo), note que só modifiquei a margem sem usar o text-indent e o br que poderiam juntos fazer um efeito semelhante

Cuidado: o uso de uma margem negativa pode acabar afetando todas as posições dos elementos seguintes

Sobreposições de margens:

As margens do eixo y tem um comportamento curioso que ao uma borda superior se encontrar com uma inferior elas se misturam, deixando apenas um dos valores da borda ter efeito, exemplo, uma margem inferior de 20px se encontrando com uma superior de 25px a distância que terá entre os elementos será de 25px e não de 45px, mas ambas ainda possuem as margens

Contudo as margens do eixo x se somam, isso quer dizer que ao duas margens se encontraram elas não se misturam, exemplo, uma margem da direita de 10px se encontrando com uma esquerda de 15px o valor do espaço entre elas será de 25px

Os dois parágrafos acima possuem o exemplo citado sobre o eixo y e este parágrafo, contém o exemplo dado sobre margens no eixo x
