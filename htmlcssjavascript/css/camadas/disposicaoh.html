Bloco

A disposição de um bloco faz com que um elemento ocupe uma linha inteira

Desta forma o elemento que for um bloco cria uma linha só para ele, sem permitir elementos aos seus lados

Mesmo se a largura do elemento for pequena, o display block continua a jogar o próximo elemento para baixo

@@@@@ Testar combinações de outras propriedades e não só valor, testar tbm um bloco dentro de um inline e por aí em diante

@@@@@ A configuração da propriedade de exibição de um elemento altera apenas a maneira como o elemento é exibido , NÃO o tipo de elemento que é. Portanto, um elemento inline com display: block;não pode ter outros elementos de bloco dentro dele.

Em linha

A disposição em linha permite um elemento continuar na linha em que é inserido, por exemplo o grifado que se fosse bloco ficaria assim, criando uma linha só para ele

Este valor "inline" não permite modificarmos seu width, nem o height, nem margens de topo ou de baixo, por este motivo cuidado ao colocar determinadas propriedades

Bloco em linha

O bloco em linha nos permite ter outros elementos ao seu lado, bem parecido com o em linha só que diferentemente, este nos permite modificar altura, largura, margens do eixo y, e tantas outras, claro que se o conteúdo for muito grande joga a próxima informação para baixo

Se um bloco em linha for maior que outro:

O maior bloco se manterá e posicionará os elementos anteriores e posteriores para baixo, tal como este elemento está fazendo

@@@@@ Para arrumar isto tente usar align-self

@@@@@ Tente deixar o bloco em cima e não aqui em baixo

Se um elemento tiver um tamanho muito grande e não for limitado ele não continuará na linha, mas será jogado para baixo, caso não esteja vendo este efeito altere o tamanho da tela

Escondendo um elemento

O display: none; faz parecer que determinado elemeto nem existe, mas na real ele existe, mas na hora de rodar a tela o elemento que estiver com ele não irá ser contando para o ajustamento das outras e "desaparece", mas ele ainda continua ali e pode realizar comandos

Deixar um elemento escondido é feito para interações da tela, não deve ser usado simplesmente para posicionar um outro elemento
