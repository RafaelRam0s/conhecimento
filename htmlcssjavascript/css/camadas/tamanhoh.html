Tamanho

Largura:

O width seleciona a largura de um elemento

Ao colocar um width fixo cuidado com o tamanho definido, pois se for muito grande por exemplo uns 500 pixels ao visualizar em dispositivos menores a página poderá ter comportamentos indesejados, como: rolagem horizontal da página ou irá dar um zoom para apresentar toda a página, deixando tudo menor

Cuidado também para caso o faça muito pequeno, pois ao fazer isso o seu conteúdo pode acabar estravazando

O mais recomendável ao ter que definir um tamanho para o elemento é definir diferentes tamanhos que só devem agir em determinados tamanhos de tela, ou uma largura que age de acordo com o tamanho de tela como o uso do valor em porcentagem

Caso não visualize um dos efeitos descritos altere o tamanho de tela

Altura:

A propriedade altura funciona de forma semelhante a da largura

Aumentar o tamanho de um elemento para dar um espaçamento maior de um conteúdo até outro não é uma boa prática já que em casos assim utilizamos a margem

Temos que tomar cuidado para não deixá-la pequena demais ao ponto de vazar conteúdo

O valor porcentagem na altura não funciona, pois o porcentagem se baseia no elemento pai para definir uma parte que ocupará a partir dele, e como não tem um valor base e sim só automático, ele não funciona, mas se definirmos para todos os elementos começando pelo body as suas alturas poderemos usar o porcentagem nele

Responsividade:

Diminua e aumente a tela para poder ver os efeitos de responsividade dos elementos a seguir:

O max significa que determinada ação, só ocorrerá até o tamanho máximo de tantos pixels da tela

O min seria o oposto, que determina a ação que só ocorre após o determinado tamanho de tela

A função @media serve para indicar que determinadas configurações definidas em seu bloco só ocorreram em um determinado tamanho de tela

Os valores max e min não dependem do @media

Caso o limite do tamanho do elemento seja menor que o espaço necessário para o conteúdo aparecer as informações do conteúdo extravazarão

Tamanho da caixa:

O box-sizing define como deve se comportar o width e o height se eles devem só se referir ao conteúdo ou ao preenchimento e borda

Se for definido o modelo de caixa de borda os tamanhos são considerados em todo o elemento

Se for definido o modelo de caixa de conteúdo os tamanhos são considerados somente no interior
