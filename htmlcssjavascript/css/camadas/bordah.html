Borda taquiográfica:

Desta forma podemos definir diversas configurações de borda para um elemento através de uma propriedade

Desta forma podemos selecionar cada parte da borda e configurar diversas propriedades para um lado específico, apesar das bordas estarem um trapézio isso acontece pela interação entre elas mas se tirarmos uma parte ficará normal

Borda arredondada

As bordas arredondadas acabam por modificar o formato geral do elemento, podendo deixar transbordar informações como os textos para fora do conteúdo se não tiver cuidado

A borda com valor dado em pixel varia entre 0-16px dando uma leve modificada nas pontas do elemento

A borda com porcentagem, permite alterar a borda como um todo deixando-a mais circular, se a altura e largura do elemento forem iguais e o valor do border-radius for 50% o elemento ficará perfeitamente redondo

Borda externa:

As bordas criam um espaçamento que acaba aumentando a distância entre outros elementos, já as bordas externas criam uma borda que não interfere, porém são mais limitados pois não podem diferenciar cada parte delas

Espaçamento do Contorno:

O contorno apresenta uma propriedade que nos permite criar um espaçamento vazio da borda até ele

Este espaço não interfere no tamanho e nem espaçamento do elemento

Quando se tem elementos com bordas se encontrando podemos definir se queremos junta-las ou deixar separada, o pequeno espaço que se forma entre duas bordas de um html pode ser explicado como o espaço que é dado no código, pois uma quebra de linha, um tab ou um espaço já são contabilizados dando um separamento de um elemento ao outro
