Ícones em lista:

Para que seja possível a sua utilização, devemos começar com uma unorded list com classe - fa-ul -, e a cada list item colocar um - span - com classe - fa-li - para que o ícone fiquem bem posicionado

  • Caso não siga a regra do fa-li isto acontece
  • Se colocar o fa-li na classe do i dará certo porém fugirá dos padrões
  • Caso não coloque o fa-ul na tagname ul os itens de lista ficarão como tópicos com um square e sem ícones
  • Adicione estilos de lista em cada item para ficar bem estilizado
  • Este é um exemplo de como daria errado pois o fa-li depende da fa-ul, logo ele não aparecerá
  • Não dependemos do fa-li, mas ficaríamos com o square atrás e um espaço de indent do texto menor

Ícones animados:

Ícones podem possuir movimentações tais como girar

Posicionamento de ícones:

Podemos colocar o ícone em diferentes ângulos com o fa-rotate-90, fa-rotate-180 e fa-rotate-270, e até inverter a ordem de direção com fa-flip-horizontal e fa-flip-vertical, se combinar ambos ângulos e direção terá uma variedade vaste de possibilidades para o posicionamento

Para podermos acrescentar uma borda nos ícones existe uma classe própria para isso a - fa-border -, agora para colocarmos ícones fixos do lado esquerdo usamos o - fa-pull-left - e na diretita o - fa-pull-right -, o que eles fazem é ser um elemento constante e conforme for escrevendo o texto eles são usados como base para saber a partir de onde o texto começa e a partir de ele termina, enquanto não for preenchido o tamanho do ícone o texto ficará por cima

Mistura de ícones

Para misturar vários ícones, use a classe fa-stack no pai, a fa-stack-1x para o ícone de tamanho regular e fa-stack-2x para o ícone maior. A fa-inverse pode ser usada como uma cor alternativa para o ícone, por padrão vem branca então cuidado. Você também pode adicionar classes de ícone maiores ao pai para controlar ainda mais o dimensionamento.

Para produzirmos este efeito também é necessário criar um conteiner geral que seria esse span, possuindo a classe fa-lg, que indica para os próximos ícones que devem ocupar todo o espaço do elemento, e o fa-stack serve para ajustar o posicionamento, já que os fa-stack acabam por utilizar propriedades como posicionamento absoluto e alinhamento de texto
Basicamente o fa-stack-1x serve para falar que aquele ícone vai se manter, e os outros ícones se basearão nele
Já o fa-stack-2x faz com que se adapte ao primeiro ícone
