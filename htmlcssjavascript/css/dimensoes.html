Tamanhos

Largura:

A partir de agora colocarei bordas e fundos nos parágrafos para que possamos enxergar os efeitos visuais

E como podemos perceber neste parágrafo sua largura está menor que o de cima

Altura:

É importante termos cuidado com os tamanhos que colocamos, pois dependendo de qual valor coloquemos o seu conteúdo pode acabar vazando, por exemplo se diminuíssemos o height deste conteúdo para 10px

Também é possível além de mandar o elemento ter apenas um valor de altura e largura, podemos colocar um limite com as propriedades de min e max

Perceba que a altura e largura se referem ao tamanho do conteúdo e não do elemento, essa anotação é importante para entendermos as propriedades a seguir

Margens

Espaçamento vazio:

Apesar da margem parecer que altera o height e width de um elemento, ele não é utilixado para isso, serve mais para criar um espaçamento de um elemento até outro

As margens laterais se somam já no caso das margens verticais de um elemento, quando encontram a margem de outro elemento elas não irão se somar e sim se misturar, por exemplo estes parágrafos que tem por padrão 16px tanto em cima quanto em baixo, então se quisermos aumentar o espaço entre eles temos que colocar um dos valores acima de 16px, agora caso queiramos diminuir a distância precisaremos modificar as margens dos dois elementos, já nas horizontais as margens se somarão

Por exemplo este que está com um espaçamento do topo maior que o convencional desta página

E este parágrafo que está centralizado, por conta das margens automáticas e do width selecionado
