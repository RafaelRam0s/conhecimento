Pseudo-classes são agregadores na chamada do elemento, são classes que só são ativados quando o usuário tiver uma determinada interação com a página

Para uma pseudo-classe funcionar é preciso que onde foi escrito as propriedades do elemento, seja escrito a pseudo-classe e seu efeito

A pseudo-classe não necessariamente ficará presa ao elemento que vc indicou, vc pode indicar mais caminhos mesmo após ter definido uma pseudo-classe
