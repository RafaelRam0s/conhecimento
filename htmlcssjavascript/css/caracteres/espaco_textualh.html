Espaçamentos de um texto:

Este é um parágrafo modelo para que possam comparar os próximos parágrafos e seus efeitos com css. Se atente aos espaços, lembrando que nem sempre precisamos usar só valores positivos para esses espaçamentos as vezes é aceito utilizar valores negativos.

Espaço entre letras:

Dando espaço, entre as letras com valor positivo.

Dando espaço, entre as letras com valor negativo.

Espaço entre palavras:

Dando espaço, entre as palavras com valor positivo.

Dando espaço, entre as palavras com valor negativo.

Espaço inicial de um texto:

Dando um espaçamento inicial em um texto, isto é, somente a primeira linha ficará mais avançada, então se um texto ficasse muito grande ou houvesse um - br - conseguiriamos ver a diferença clara deste efeito.
Caso não tenha conseguido observar este efeito diminua e aumente a tela.

Espaço entre as linhas:

Dando espaço entre as linhas de um texto, para conseguirmos ver isto, é preciso de um parágrafo com ao menos duas linhas então para conseguir este efeito que descrevi, estou enrolando sobre como é possível visualizar o efeito.
Se bem que a tagnema - br - pode deixar isto ocorrer mais facilmente, bem estes são os efeitos entre as linhas:

Se deixar um valor muito pequeno tem-se um problema de mal conseguir visualizar.
Lembrando que esta propriedade não aceita valores negativos

Regras de espaço

O html por padrão não conta espaços a mais dados, nem considera os tabs que damos, também não considere enters dados no meio do código e ele ajusta uma quebra de linha automática quando o conteúdo está prestes a vazar do elemento, mas estas regras de leitura podem ser modificadas criando diversas combinações destas regras.

Altura de um elemento na linha

Temos como deixar um elemento mais alto ou mais baixo na linha dependendo de como configuramos suas propriedades no css, possibilitando-nos de fazer representações de exponenciações e de elementos químicos.

Por exemplo o 2 de um número x ao quadrado (X2)

E/ou por exemplo o 2 do elemento químico da água (H2O)
