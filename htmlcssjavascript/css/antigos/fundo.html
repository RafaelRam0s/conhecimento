Fundos

Mudando a cor do fundo

Ao colocar uma cor de fundo verifique se não existe outros elementos da mesma cor, pois caso as cores coincidão, poderá ter uma impressão de que algo não está funcionando ou que sumiu, quando na verdade está "camuflado"

Opacidade

Caso deseje fazer uma transparência do fundo, pode-se fazer deste jeito. Cuidado pois ao tentar aumentar depois a sua opacidade pode dar errado por exemplo a palavra a seguir que independente do valor que eu coloque não fica como era antes: Inconstitucionalissimamente ao valor que eu coloque, pórem para diminuir ainda mais é possível sem ter muitos problemas além de ler essa frase fraca.

Imagem de fundo

Repare que ao colocarmos uma imagem de fundo ela pode acabar por atrapalhar a leitura do texto, então se atente se tudo estará legível

Podemos fazer mistura de fundos, mas é preciso tomar alguns cuidados como; se o fundo possue espaços vazios para que uma imagem permita visualizar partes da outra e se a ordem de precedência no css está correta

Podemos colocar a imagem tanto por link quanto por referência na pasta, se não aparecer uma imagem diferente que a do conteiner anterior que é a imagem de 200x200 é porque o link do qual tirei essa imagem não serve mais

Tipos de posição

Favor testar todas as suas formas de posições na folha de estilo (css externo) pois ficaria mais feio do que já está este site se eu tentasse demonstrar cada uma, e a propósito use uma mistura com o background-repeat para poder ter noção de diversos efeitos

Tipos de fixação do fundo

Esse formato é o que vem por padrão e o que ele faz é deixar o fundo junto do elemento

Já desse jeito conforme você vai passando a página o fundo fica fixo lá atrás

Agora a diferença desse é que a sua imagem vai aparecendo conforme o seu elemento é rodado
