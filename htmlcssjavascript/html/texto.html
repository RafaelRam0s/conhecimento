Formatações de texto:

Existem algumas formatações de texto que ajudam o usuário a prestar mais atenção em alguns pontos de leitura da página, como o negrito, itálico ... e por aí vai, seguem exemplos de como fazer esses efeitos;

Texto em negrito, porém sem importância semântica, isto é, será negrito apenas para a visualização do usuário, mas não para o navegador

Texto em negrito com importância semântica, como estamos com um estilo de texto de cor branca pode estar ruim a visualização do seu efeito, se atente a esses fatos na sua página

Texto em itálico, mas sem importância semântica, a tagname "i", também serve para inserir ícones, será explicado futuramente neste tutorial

Texto em itálico, com importância semântica

Texto grifado

Texto pequeno

Texto delete deletado e inserido, usa-se o del para marcar textos que foram removidos, e ins para textos que foram inseridos, não os utilize simplesmente para colocar um traço nas palavras pois ambos possuem uma função para leitores de tela

Texto subscrito usado na maioria das vezes com números, por exemplo: H2O, log10100 = 2

Texto superescrito usado em muitos casos para contas matemáticas como: x23

Quebra de linha por tagname:

É possível quebrar a linha do conteúdo para poder continuar no mesmo elemento e ainda apresentar informações, vou fazer dois parágrafos iguais para poder comparar este efeito. O 1° será um parágrafo normal. O 2° será um parágrafo com quebras de linha após os pontos finais. Caso não perceba seus efeitos altere o tamanho da tela

Uso de aspas:

Para citações, vulgo o aspas, usamos o quotation e Assim que tivermos quotation dentro de outro quotation será feita aspas simples, se houvesse três ou mais quotations dentro um do outro, a partir do segundo quotation só serão adicionadas aspas simples

Citações grandes:

Caso tenho um texto muito grande retirado de outra fonte o mais indicado é fazer um bloco de aspas para referencia-lo, normalmente é mais visto este tipo de efeito em livros de filosofia que citam o argumento de um filósofo, combine o blockquote com o quotation para ter um efeito bacana igual a este que fiz

Abreviações:

Ao mercionarmos palavras abreviadamente, é recomendado que a marquemos com a tagname - abbr - e adicionando o atributo - title - podemos explicar o seu significado por extenso, e para o usuário visualizar o que seria a abreviação bastaria passar com o mouse em cima da palavra, seguem exemplos, passe com o mouse sobre eles para poder ver o efeito que mencionei: PC, TV, html...

Citações:

Ao mencionarmos um título de uma obra tipo a Monalisa ou a música Highway to Hell da banda AC/DC, é correto marca-los com a tagname apropriada

Código de computador:

E caso estejamos falando de algum código de computador utilizamos a tagname - code - que tem o estilo no css da seguinte maneira: code {font-family: monospace;}, normalmente quando o utilizamos tambem colocaos a tagname - pre - para que possa mostrar um código mais rápido e fácil, por exemplo:
