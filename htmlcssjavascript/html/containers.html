Explicando containers

O header deve ser a primeira informação da sua página, onde por padrão contem uma imagem de sua marca com link para página inicial, barra de menu para navegação, um botão de logar (caso tenha na sua página métodos de inscrição), uma aba de pesquisar, informações rápidas de contato, redes sociais. Enfim qualquer coisa que ajude a navegar nas suas páginas

O container nav serve para armazenar links de navegação, não necessariamente estará dentro do header

E nas sections, é onde terão os conteúdos se sua página

Detalhes e sumário:

Informação Lateral:

Quando precisamos dividir uma página no meio para demontrar duas informações em paralelo podemos usar este container, tábem é possível usá-lo como uma lateral de navegação

Rodapé

No rodapé de uma página devem ser colocadas informações para contato, informações da empresa ou criador, dúvidas frequentes, termos de uso, privacidade, a parte mais burocrática de um site, assim como podem conter auxílios de navegação e páginas parceiras. Ao fazer um rodapé deixe-o no máximo ocupando metade da tela, senão pode ficar extranha a visualizção para um usuário
