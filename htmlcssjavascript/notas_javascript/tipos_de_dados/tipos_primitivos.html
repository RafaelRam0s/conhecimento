Tipos primitivos

Strings

String é qualquer dado que esteja contido no meio de aspas duplas, simples ou crases

Numbers

Number é qualquer número, seja ele inteiro, como; 1, 89, 0, -105, -2 ou racional (onde a vírgula da regra matemática brasileira é trocada por ponto final), como; 0.8, 3.4, -52.0

Infinity

NaN

Not a number - não é um número

Booleanos

Boolean é qualquer ação lógica que pode retornar valor true ou false

Typeof
