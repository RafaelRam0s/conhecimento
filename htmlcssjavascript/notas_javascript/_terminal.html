Terminais agem como consoles

Para se usar o terminal siga os paços:

1° Abra o terminal no vscode é ctrl+shift+'

2° Escreva node, assim ele usará o terminal do node.js

A partir disso você pode escrever códigos de javascript, tipo var = x;

Ao finalizar o que estava fazendo escreva exit aí sairá do terminal node.js, e escreva de novo exit para fechar o terminal que está ativo

Cuidado para não abrir um monte de terminais e estar fechando de forma errada, o jeito é fazer o que foi descrito no parágrafo acima
